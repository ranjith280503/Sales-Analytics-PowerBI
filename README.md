# 📊 Sales Performance Analytics Dashboard

## 📌 Project Overview

This project is an end-to-end Power BI Sales Performance Analytics Dashboard built using a **Galaxy Schema (Fact Constellation Schema)** data model. The dashboard provides actionable insights into sales performance, campaign effectiveness, inventory management, promotion coverage, and order processing across multiple business dimensions.

---

## 🏗️ Data Model

The project follows a **Galaxy Schema**, where multiple fact tables share common dimension tables for efficient analysis and reporting.

### Dimensions

- dim_campaign
- dim_customer
- dim_date
- dim_geo
- dim_order_flags

### Fact Tables

- fact_campaign_spend
- fact_inventory
- fact_orders_to_process
- fact_promotion_coverage
- fact_sales
- fact_sales_targets

## 🎯 Business Objectives

- Monitor overall sales performance.
- Analyze campaign spending effectiveness.
- Track sales against targets.
- Monitor inventory availability.
- Evaluate promotional coverage.
- Track pending orders for processing.
- Analyze customer and geographical performance.
